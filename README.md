# Meine Coole Website

This repository is for my cool website using React and Tailwind CSS.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/schneidi2012/meine-coole-website.git
   ```

2. Change directory into the project:
   ```bash
   cd meine-coole-website
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Run the project:
   ```bash
   npm start
   ```

## Available Scripts

- `npm start`: Runs the app in the development mode.
- `npm test`: Launches the test runner.
- `npm run build`: Builds the app for production.

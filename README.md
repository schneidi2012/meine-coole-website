# Meine Coole Website 🚀

A visually stunning, interactive website built with **React**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## ✨ Features

- 🎨 **Modern Design** — Bold gradients, glassmorphism effects, and beautiful typography
- ⚡ **Smooth Animations** — Framer Motion powered animations and transitions
- 🌙 **Dark/Light Mode** — Theme toggle with seamless switching
- 📱 **Fully Responsive** — Mobile-first design that looks great on all devices
- 🎯 **Interactive Sections** — Hero, Features, Stats, Demo, Testimonials, CTA

## 🛠️ Tech Stack

- **Vite** — Lightning-fast build tool
- **React 18** — UI library
- **TypeScript** — Type-safe development
- **Tailwind CSS v3** — Utility-first styling
- **Framer Motion** — Production-ready animations
- **Lucide React** — Beautiful icons

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/schneidi2012/meine-coole-website.git
   cd meine-coole-website
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📦 Available Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run preview` — Preview production build
- `npm run lint` — Run ESLint

## 📁 Project Structure

```
src/
├── components/

│   ├── Stats.tsx         # Animated counter statistics
│   ├── Demo.tsx          # Interactive code demo
│   ├── Testimonials.tsx  # Testimonial carousel
│   ├── CTA.tsx           # Call-to-action section
│   └── Footer.tsx        # Site footer
├── App.tsx               # Root component with theme management
├── main.tsx              # Entry point
└── index.css             # Global styles with Tailwind directives
```
